# Документация HabitFlow

HabitFlow — это монорепозиторий с backend на FastAPI, frontend на Vue и PostgreSQL. Документация решает две задачи:

- Новый разработчик понимает систему менее чем за час.
- Старший инженер может провести архитектурный аудит решений и рисков.
